# 台股專業分析系統 📈

即時股價監控 + 技術分析 + AI 三方票決

## 功能特色

- 📊 **多股即時監控** - 4宮格同時監控8檔股票
- 🧠 **智能模式** - 交易時間(09:00~13:30)自動啟動
- 📱 **iPad 平板模式** - 12.9吋專用全螢幕監控
- 🔥 **類股熱力圖** - 28個產業漲跌一覽
- 📈 **技術指標** - MA/MACD/RSI/KD/布林通道
- 🤖 **AI 三方票決** - Claude + Gemini + GPT 綜合建議

## 部署方式

1. Fork 此倉庫
2. 進入 Settings → Pages
3. Source 選擇 `main` branch
4. 等待部署完成，訪問 `https://你的帳號.github.io/taiwan-stock-analyzer`

## iPad 使用

1. Safari 開啟網址
2. 點擊右下角「📱 iPad模式」
3. 分享 → 加入主畫面
